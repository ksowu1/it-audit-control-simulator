Installation
============

Follow these steps to install and run **IT Audit Control Simulator**.

Requirements
------------

- Python 3.10+
- pip
- Git

Clone Repository
----------------

.. code-block:: bash

   git clone https://github.com/ksowu1/it-audit-control-simulator.git
   cd it-audit-control-simulator

Install Dependencies
--------------------

.. code-block:: bash

   pip install -r requirements.txt

Run the Simulator
-----------------

.. code-block:: bash

   python main.py
